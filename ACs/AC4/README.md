# AC4 – Deploy de Aplicações Django na AWS

## Curso

Engenharia de Computação

## Disciplina

Big Data & Cloud Computing

## Período

2026.1

## Descrição

Atividade avaliativa sobre deploy e gerenciamento de aplicações Django REST na AWS utilizando serviços gerenciados de computação em nuvem. A atividade aborda Elastic Beanstalk, integração com Amazon RDS e Amazon S3, escalabilidade automática, gerenciamento de variáveis de ambiente e persistência de arquivos em ambientes distribuídos.

## Objetivos de Aprendizagem

* Compreender o funcionamento do AWS Elastic Beanstalk
* Implantar aplicações Django REST em ambiente cloud
* Entender conceitos de Auto Scaling e Load Balancing
* Configurar variáveis de ambiente em aplicações Django
* Integrar aplicações com Amazon RDS
* Configurar armazenamento persistente utilizando Amazon S3
* Aplicar boas práticas de arquitetura cloud e persistência de dados

## Tecnologias e Serviços AWS Abordados

* AWS Elastic Beanstalk
* Amazon EC2
* Elastic Load Balancer (ELB)
* Auto Scaling Groups
* Amazon RDS MySQL
* Amazon S3
* django-storages
* boto3
* Django REST Framework
* Variáveis de ambiente
* Deploy cloud
